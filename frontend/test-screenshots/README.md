# 前端测试截图目录

## 📁 目录说明

本目录存储历史数据统计分析工具前端UI测试过程中生成的所有截图文件。

## 📋 截图目录结构

### 🎯 NLP UI测试截图
- **`nlp-ui-test-screenshots/`** - NLP前端UI测试截图目录
  - 功能测试截图
  - 错误状态截图
  - 响应式设计测试截图

### 📸 通用测试截图
- **`screenshots/`** - 通用测试截图目录
  - 页面加载截图
  - 交互过程截图
  - 最终结果截图

## 📷 截图类型说明

### 功能测试截图
- **页面加载**: 应用启动和页面初始化
- **导航测试**: 页面跳转和菜单操作
- **数据输入**: 表单填写和数据提交
- **结果展示**: 分析结果和图表显示

### 错误状态截图
- **加载失败**: 页面或资源加载错误
- **功能异常**: 功能执行失败状态
- **网络错误**: 网络连接问题
- **数据错误**: 数据格式或内容错误

### 响应式设计截图
- **桌面视图**: 1920x1080分辨率
- **平板视图**: 768x1024分辨率
- **手机视图**: 375x667分辨率

- **颜色**: 真彩色

### 截图时机
- 页面加载完成后
- 功能执行前后
- 错误发生时
- 测试完成时

### 自动截图触发
```javascript
// 页面加载完成
await page.waitForLoadState('networkidle');
await page.screenshot({ path: 'page_loaded.png' });

// 功能执行后
await page.click('#analyze-button');
await page.waitForSelector('.results');
await page.screenshot({ path: 'analysis_complete.png' });

// 错误发生时
try {
  // 测试代码
} catch (error) {
  await page.screenshot({ path: 'error_state.png' });
}
```
